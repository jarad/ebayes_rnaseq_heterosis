\begin{frame}
We might try to adjust the moment estimators using the relation:\\
\[Var(\hat{\delta_g}) = E(Var(\hat{\delta_g}|\delta_g)) + Var(E(\hat{\delta_g}|\delta_g))\]
Rearranging yields
\[ Var(\delta_g) \approx Var(E(\hat{\delta_g}|\delta_g)) = Var(\hat{\delta_g}) - E(Var(\hat{\delta_g}|\delta_g))\]
Which we might estimate by
\[Var(\delta_g) \approx \widehat{Var}(\hat{\delta_g}) - \widehat{E}(SE(\hat{\delta})^2)\]

To estimate $\widehat{E}(SE(\hat{\delta})^2)$, we can use estimated inverse information from genewise MLE fitting conditional on edgeR estimates of $\psi_g$.
